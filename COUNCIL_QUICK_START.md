### Session Takes Too Long
- **Normal**: 1-3 minutes is expected
- **Fix**: Use Quick mode or faster model

### All Proposals Vetoed
- **Cause**: Strategist found all solutions unethical
- **Fix**: Reframe question to allow ethical options

- **Symptom**: Errors or poor responses
- **Fix**: Use models that support system prompts (GPT-4, Claude, Llama)
